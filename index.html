Neural network embeddings


In recent years, neural networks have gained popularity as a method for embedding high-dimensional categorical data into more useful numerical representations. This process forms a mapping from the categorical space to a continuous space where similar categories are mapped to similar locations.

Many of these techniques have been developed for Natural Language Processing (NLP), where words are mapped to a vector space that encodes the relationships and meaning of words. These techniques can also be applied beyond natural language to other sequence data.
Embeddings for cybersecurity


By observing the sequence of programs running on a computer, deep learning can develop a meaningful mapping of those programs. Inspired by techniques in Natural Language Processing, these mappings encode an abstract understanding of the uses and similarities of programs.

This is a visualization of one of these abstract representations, where each point corresponds to a program. Models learn that programs with similar applications occur in similar contexts, and is able to cluster them together.
Embedding phase



Starting from random, embedding vectors are learned by observing the pairs of processes that occur together. As more data is passed through the model, processes are organized into meaningful clusters.

Endpoint behavior encoding

Various forms of data are collected from the endpoint and analyzed in real-time. Values for a set of endpoint metrics collected over time are analogous to the pixel intensities of an image of a face; different combinations of collected data can be used to uniquely identify the activity of a specific user or machine.
Model training

Using machine learning techniques borrowed from facial recognition, accurate models of user behavior and machine activity can be learned. More importantly, these profiles can be used to determine when an unknown entity accesses a priviledged machine or system.
